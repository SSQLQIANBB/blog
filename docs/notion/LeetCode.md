# 一、双指针总纲

> **所有双指针题，99% 都落在这四类之一**

| 大类     | 指针运动方式  | 本质      |
| ------ | ------- | ------- |
| ① 快慢指针 | 同向，不同速度 | 过滤 / 去重 |
| ② 左右指针 | 两端向中间   | 有序 + 对撞 |
| ③ 滑动窗口 | 同向，窗口伸缩 | 区间最优    |
| ④ 多指针  | ≥3 指针   | 排序 + 固定 |

---

# 二、① 快慢指针（同向）

## 🔑 适用场景

* 原地修改数组
* 删除 / 去重
* 链表环

---

## 典型模板

```js
let slow = 0;

for (let fast = 0; fast < n; fast++) {
  if (满足条件) {
    nums[slow] = nums[fast];
    slow++;
  }
}
```

---

## 高频题目（必刷）

| 题号     | 题名        | 核心     |
| ------ | --------- | ------ |
| LC 27  | 移除元素      | 覆盖     |
| LC 26  | 删除排序数组重复项 | 相邻比较   |
| LC 80  | 删除重复项 II  | slow-2 |
| LC 283 | 移动零       | 非零覆盖   |
| LC 141 | 环形链表      | 快慢追    |
| LC 142 | 环入口       | 数学     |

---

## 代表例子：LC 80

```js
if (nums[fast] !== nums[slow - 2]) {
  nums[slow++] = nums[fast];
}
```

👉 **slow-2 = 已写入的倒数第 2 个**

# 三、② 左右指针（对撞指针）

## 🔑 适用场景

* **有序数组**
* 求和 / 回文 / 盛水

---

## 模板

```js
let l = 0, r = n - 1;

while (l < r) {
  if (条件) l++;
  else r--;
}
```

---

## 高频题目

| 题号     | 题名      | 核心      |
| ------ | ------- | ------- |
| LC 167 | 两数之和 II | 升序      |
| LC 15  | 三数之和    | 固定 + 对撞 |
| LC 11  | 盛最多水    | 短板      |
| LC 125 | 验证回文串   | 忽略字符    |
| LC 344 | 反转字符串   | swap    |

---

## 代表例子：两数之和 II

```js
while (l < r) {
  const sum = nums[l] + nums[r];
  if (sum === target) return [l+1, r+1];
  sum < target ? l++ : r--;
}
```

# 四、③ 滑动窗口（最容易混）

## 🔑 适用场景

> **连续子数组 / 子串 + 最值**

---

## 模板（牢记）

```js
let left = 0;

for (let right = 0; right < n; right++) {
  // 扩张窗口

  while (不满足条件) {
    // 收缩窗口
    left++;
  }

  // 更新答案
}
```

---

## 高频题目（超级高频）

| 题号      | 题名       |
| ------- | -------- |
| LC 209  | 长度最小子数组  |
| LC 3    | 无重复字符子串  |
| LC 76   | 最小覆盖子串   |
| LC 567  | 字符串排列    |
| LC 438  | 找所有字母异位词 |
| LC 1004 | 最大连续 1   |

---

## 代表例子：LC 3

```js
while (set.has(s[right])) {
  set.delete(s[left++]);
}
set.add(s[right]);
```

for i:

# 六、双指针 vs 其他算法 一眼判断

| 题目特征    | 算法   |
| ------- | ---- |
| 原地修改数组  | 快慢指针 |
| 有序 + 求和 | 左右指针 |
| 连续区间最值  | 滑动窗口 |
| 层 / 最近  | BFS  |
| 所有路径    | DFS  |

---

# 七、10 秒强制判断流程

```
1️⃣ 是否连续区间？
  → 是：滑动窗口

2️⃣ 是否原地删除 / 覆盖？
  → 是：快慢指针

3️⃣ 是否有序数组 + 求和？
  → 是：左右指针

4️⃣ 是否 ≥3 数？
  → 排序 + 多指针
```
